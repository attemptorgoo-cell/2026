# 程序计数器 PC

*从0x0000 0000开始往下读取指令，依次加4。
#指令
*其中指令是32位，0x表示十六进制，一位可以转化为四个2进制。
*PC是以字节计算，一个字节8位，4 * 8 = 32。

#地址
*0x0000 0000是主CPU地址，0x1000 0000可以放其他的，比如串口。
# 寄存器堆

*存放指令，存放数据。

寄存器堆不占用地址，x0始终为0，其余有x1-x31。
