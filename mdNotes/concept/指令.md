每条指令都是 **32 位**，不同类型（R/I/S/B/U/J）有不同字段，但核心都是：

|字段|位宽|含义|
|---|---|---|
|opcode|7|指令类型（如 ALU、load、store、jump）|
|rd|5|目的寄存器|
|funct3|3|指令子类型（比如 add/sub，beq/bne）|
|rs1|5|源寄存器1|
|rs2|5|源寄存器2|
|funct7|7|扩展功能码（比如 add/sub 区分）|
|imm|可变|立即数（I/S/B/U/J 类型不同）|

| 类型              | opcode            | 解析逻辑                                                    |
| --------------- | ----------------- | ------------------------------------------------------- |
| R型（寄存器运算）       | 0110011           | ALU 指令：通过 funct3 + funct7 区分 add/sub/and/or/xor/sll/sra |
| I型（立即数 / load）  | 0000011 / 0010011 | funct3 区分 addi / lw / jalr                              |
| S型（存储）          | 0100011           | funct3 区分 sw / sb / sh                                  |
| B型（分支）          | 1100011           | funct3 区分 beq / bne / blt / bge                         |
| U型（lui / auipc） | 0110111 / 0010111 | 只用 opcode + rd + imm                                    |
| J型（jal）         | 1101111           | 只用 opcode + rd + imm                                    |
